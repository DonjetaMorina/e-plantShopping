# e-plantShopping — Paradise Nursery

Welcome to **e-plantShopping**, the official repository for the **Paradise Nursery** front-end web application. This project was developed as a final assignment to showcase a dynamic React and Redux-driven e-commerce user experience.

## Project Overview

**Paradise Nursery** is a sleek, responsive online plant shopping application designed for houseplant enthusiasts. The interface allows users to seamlessly explore a diverse selection of plants, organize choices by category, and manage a real-time shopping cart. 

### Key Features Included:
* **Landing Page:** An elegant welcome screen featuring a lush background image, company overview, and an interactive "Get Started" button.
* **Product Listing Page:** A structured directory grouping at least six unique houseplants across three distinct categories, complete with dynamic "Add to Cart" state management.
* **Dynamic Global Header:** A persistent navigation bar tracking the total item quantity dynamically via a live shopping cart badge.
* **Shopping Cart Page:** A comprehensive cart breakdown leveraging Redux state to handle real-time subtotal calculations, item incrementation/decrementation, item deletion, and checkout flows.

## Tech Stack
* **Frontend Framework:** React.js (JSX)
* **State Management:** Redux Toolkit (`CartSlice.jsx`)
* **Styling:** CSS3 (`App.css`)
* **Deployment:** GitHub Pages
